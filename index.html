Introduction

Problem Statement: Aquarium pump failures pose significant risks to ornamental fish survival due to oxygen depletion. Traditional monitoring systems lack automated failover capabilities and real-time remote notification features.

Solution Overview: This study presents an IoT-based automatic aerator control system that employs a waterflow sensor to detect real-time water circulation and activates a backup aerator when flow interruption occurs.

Key Benefits:

  • 2.72s average failover response time
  • Real-time remote monitoring via Blynk app
  • Automatic backup aerator activation
  • Cost-effective IoT solution
  • Reduces manual oversight requirements

System Components:

  • ESP8266 Microcontroller: Central processing unit with Wi-Fi capability
  • Water Flow Sensor: Detects real-time water circulation (40-800 GPH range)
  • Relay Module: Controls backup aerator activation
  • Backup Aerator: Emergency oxygen supply system
  • Blynk App: Remote monitoring and control interface

Implementation:

The system continuously monitors water flow through the sensor. When flow drops below threshold, the ESP8266 triggers the relay to activate the backup aerator while sending real-time notifications via the Blynk mobile application.
